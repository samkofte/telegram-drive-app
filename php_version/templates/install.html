Veritabanı bilgilerinizi giriniz. Kurulum sihirbazı otomatik olarak veritabanını oluşturacak ve tabloları kuracaktır.
Uygulamanın çalışabilmesi için bir Telegram Bot Token'ı ve hedef Chat ID (Logların ve dosyaların depolanacağı grup/kanal ID) gerekmektedir.
Virgülle ayırarak birden fazla bot token ekleyebilirsiniz. Bu sayede yüklemeler dağıtılır.
Dosyaların depolanacağı Telegram kanalı veya grubunun ID'si (Botun o grupta yetkili olması gerekir).
E-posta bildirimleri ve şifre sıfırlama için isteğe bağlı Resend API anahtarı.
Yönetici paneline giriş yapabilmeniz için yönetici hesabınızı oluşturun.
